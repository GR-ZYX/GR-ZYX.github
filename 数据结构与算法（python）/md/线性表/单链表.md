## 单链表-Python实现

### 定义表节点类：`LNode`

```
class LNode:
    # __init__() 构造函数
    def __init__(self, elem, next_=None):
        self.elem = elem  # 节点存放元素
        self.next = next_  # 下一个节点
```



### 定义单链表类：`LList`

```
class LList:

    def __init__(self):
        self._head = LNode(None)
```



#### 判断链表是否为空

```
def is_empty(self):
    return self._head.next is None
```
